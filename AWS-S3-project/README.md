# AWS S3 Static Website - Beginner Practice Project

This project is designed to help beginners **learn how to host a static website on AWS S3** using the AWS Management Console.

## Project Overview
- Host a simple static website using AWS S3
- Understand S3 bucket creation, public access, and static website hosting
- Learn the basics of AWS cloud storage

## Files
- `index.html` - Main page of the website
- `error.html` - Error page for 404 not found
- `screenshots/` - Optional screenshots to guide beginners visually

## Steps to Follow
1. Create an S3 bucket in AWS Console
2. Upload `index.html` and `error.html`
3. Enable static website hosting
4. Apply a bucket policy for public access
5. Open the website using the bucket website endpoint
