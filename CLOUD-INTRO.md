                                                          INTRODUCTION TO CLOUD COMPUTING
                                                        ************************************
  * Cloud computing is defined as on demand delivery of the IT services over the internet,
  * it provids pay-as-you-go service globally,
  * it means we don't have to buy instead we can rent those infrastructures accourding to our need.
  * it has changed the traditional IT structure and also provide the most flexiability and scalibility.

  ## Types of cloud model.
  1. Public Cloud:
  * A third-party provider owns the resources, which are distributed online among several companies. They are cost effective and scalable.
     Example: AWS, Azure, GCP etc.
     
  3. Private Cloud:
  * The infrastructure is allocated to a particular organization due to which all the control related  to security are under one hand.

  4. Hybrid Cloud:
  * This is the combination of both public and private cloud model and hence can alow the users to enjoy the cons of both cloud system.

## Tyeps of cloud service model.
1. IaaS
* IaaS stands for Infrastructure as a service.
* it offers core computing infrastructure, such as virtual machines and storage, on demand.
   Example: AWS, Google Cloud, and Microsoft Azure

2. SaaS
* SaaS stands for software as a service.
* It delivers applications over the internet on a subscription basis.
   Example:  Google Docs, and Office 365.


4. PaaS
* PaaS stands for platform as a service.
* Provides a platform for developing, running, and managing applications without the complexity of building and maintaining the infrastructure.
   Example: AWS Elastic Beanstalk, and Windows Azure.


Note: Many esteemed IT organization usese AWS as cloud service such as Netflix and Spotify.
