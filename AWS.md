                                                          AMAZON WEB SERVICE (AWS)
                                                      *********************************

  * Amaon Web Services (AWS) is one of the most popular cloud provding company which provides numbers of
    IT infrastructures and services to the customers that helps them to build and run applications
    on the internet without actually buying the physical servers.

  * It is cost effective, scalable, highly secured and globally available along with wide range of services.
    From hosting websites to AI and also databases, storage , Iot along with over 200 services.
----------------------------------------------------------------------------------------------------------------------------------------    
                                         Mostly used AWS Services 
                          *********************************************************
1. Amazon EC2 (Virtual Servers) :
* EC2 provides virtual computers in the cloud.
* Used for: hosting websites, apps, backend systems, game servers.
* Real-world example: Netflix uses EC2 to stream movies.
  
2. Amazon S3 (Cloud Storage):
* S3 is used to store files like images, videos, backups, logs, etc.
* It is very reliable and scalable.
* Real-world example: Instagram-like apps store user photos in S3.

3. AWS Lambda (Serverless):
* Lambda lets you run code without managing servers. You only pay for the time your code runs.
* Real-world example: Slack and Pinterest use Lambda for background tasks.

4. Amazon RDS (Managed Database):
* RDS provides ready-to-use SQL databases like MySQL, PostgreSQL, etc.
* Real-world example: E-commerce websites use RDS to manage customer orders.

5. Amazon DynamoDB (NoSQL Database):
* A very fast database used for real-time applications and high traffic.
* Real-world example: Gaming leaderboards and shopping carts use DynamoDB.

6. Amazon VPC (Virtual Private Cloud):
* VPC helps you create your own secure network in AWS.
* All major apps use VPC for privacy and security.

8. Amazon IAM (Identity & Access Management):
* IAM manages who can access what in your AWS account.
